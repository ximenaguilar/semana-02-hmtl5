# TechStore

Proyecto desarrollado para el curso de Desarrollo Web.

## Descripción

TechStore es una página web creada utilizando HTML5. El objetivo del proyecto es aplicar la estructura semántica de una página web, incluyendo navegación, secciones, formularios, tablas, listas e imágenes.

## Estructura del proyecto

```
semana-02-html5/
│
├── index.html
├── pages/
│   ├── productos.html
│   └── contacto.html
├── assets/
│   ├── img/
│   ├── css/
│   ├── js/
│   ├── audio/
│   └── video/
└── README.md
```

## Tecnologías utilizadas
